# nconfig
Installer for dotfiles and other configurations for linux.

## Installation
### Prerequisites
```
$ pacman -S python-pip
```

```
$ apt install python3-pip
```

```
$ yum install python-pip
```

### Install:
```
$ git clone https://github.com/noahjutz/nconfig
$ cd nconfig
$ pip install .
```

### Execute:
```
$ nconfig --help
```
